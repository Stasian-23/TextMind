TextMind — Анализатор тональности отзывов Steam

Нейросетевой классификатор текста + интерактивное веб-приложение

TextMind — это проект, который анализирует тональность пользовательских отзывов из Steam.
Он включает в себя

подготовка данных,

обучение модели,

визуализация метрик,

развертывание Gradio-интерфейса,

Основной функционал
Обработка текста

Очистка текста (нижний регистр, удаление символов)

Токенизация

Построение словаря

Векторизация отзывов

Модель

Полносвязные слои

Dropout для борьбы с переобучением

Функция потерь

Графики

График изменения Loss

График Accuracy

 Интерфейс

Интерактивный веб-интерфейс на Gradio:

Ввод текста пользователем

Мгновенный вывод тональности (положительный / отрицательный)

Отображение вероятности

Установка и запуск
1. Клонирование репозитория
git clone https://github.com/Stasian-23/TextMind
cd TextMind

2. Установка зависимостей
pip install -r requirements.txt

Открой файл:
Анализатор_тональности_отзывов_Steam__!.ipynb

3. Запуск Gradio-приложения внутри

В конце:

app.launch(share=True)
